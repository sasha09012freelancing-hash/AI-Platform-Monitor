3. Детали по платформам

Vertex AI: Google активно развивает возможности работы с данными, внедряя LLM-функции прямо в SQL-запросы для кратного ускорения аналитики. Продолжается активное продвижение платформы как лидера в корпоративной разработке ИИ-приложений.

Сигналы: Интеграция LLM в SQL для ускорения запросов, Признание лидерами в Gartner Magic Quadrant

AWS Bedrock: Платформа делает упор на Enterprise-безопасность, внедряя контроль доступа к документам и защищенные взаимодействия между агентами (A2A). Особое внимание уделено ограничению прав агентов через политику корпоративных браузеров.

Сигналы: Ограничение доступа к документам S3, Сотрудничество с Cisco для безопасности A2A

Azure Foundry: Релиз Foundry Local 1.1 и GA-статус ключевых инструментов упрощают локальную разработку ИИ-агентов. Платформа расширяет библиотеку моделей и инструменты для оценки (batch evaluations).

Сигналы: Релиз Foundry Local 1.1, Апрельский дайджест обновлений

AutoGen: Сообщество сфокусировалось на создании защитных механизмов для multi-agent систем, включая скоринг доверия и защиту памяти от отравления. Активно обсуждаются протоколы для обеспечения долговечности агентской работы.

Сигналы: Предложение по скорингу доверия агентов, Инструмент OWASP для защиты памяти агентов

AgentForce: Salesforce трансформирует разработку, переводя ее в формат «agent-first» с поддержкой метаданных и CI/CD. Внедрена возможность интеграции кастомной логики Apex как MCP-инструментов.

Сигналы: Жизненный цикл метаданных для агентов, Публикация Apex как MCP-инструмента

LangGraph: Произошел переход к официальным стабильным версиям (v1.2.0), закрепляющим API для управления состоянием и стриминга событий. Улучшена поддержка работы с базами данных для чекпоинтов.

Сигналы: Релиз LangGraph 1.2.0, Обновление событийно-ориентированного стриминга в LangChain

n8n: Получение стратегических инвестиций от SAP подтверждает лидерство n8n в enterprise-автоматизации. Интеграция с Joule Studio делает визуальную оркестрацию агентов доступной для корпоративных пользователей SAP.

Сигналы: Инвестиции SAP в n8n, Партнерство для визуальной оркестрации

5. Выводы для команды

  1. Фокус индустрии смещается на стандартизацию протоколов обмена данными (MCP) и безопасности взаимодействия между агентами. SL2 AI Platform необходимо активно развивать собственную реализацию MCP-контрактов для сохранения совместимости с экосистемой.
  2. Инвестиции SAP в n8n доказывают запрос рынка на Low-code визуализацию агентских потоков внутри существующих enterprise-систем; это сигнал к расширению возможностей No-code интерфейса SL2 AI Platform.
  3. Безопасность памяти агентов стала критическим вектором атак. Внедрение функций контроля доступа и очистки контекста (на уровне tenant) в SL2 AI Platform должно стать приоритетной задачей для соответствия enterprise-стандартам.
