3. Детали по платформам

Vertex AI: На неделе Google Cloud анонсировал fully-managed remote MCP server для Gemini Enterprise Agent Platform (DP). Это усиливает критерий Unified API + контракт, так как MCP (Model Context Protocol) становится стандартизированным способом взаимодействия агентов с внешними системами (CP). Для SL2 AI Platform это давление на GAP G2 (Agent Protocol несовместим с A2A), так как MCP может стать де-факто стандартом для межагентных коммуникаций. Однако Vertex AI остаётся managed-решением, что сохраняет наше преимущество в self-host для enterprise-клиентов (SP).

Сигналы: Gemini Enterprise Agent Platform remote MCP server

AWS Bedrock: AWS выпустил Claude Sonnet 5 и представил AG-UI протокол для generative UI в агентах на базе AgentCore (DP). AG-UI усиливает Unified API + контракт, так как предоставляет стандартный способ рендеринга UI для агентов (CP). Также Bedrock улучшил multi-tenancy через managed entitlements для распределения доступа к моделям между аккаунтами (CP). Для SL2 AI Platform это давление на GAP G4 (нет cost tracking per org/agent) и G6 (своя мультипровайдерная обвязка вместо LiteLLM), так как Bedrock предлагает более зрелые инструменты для enterprise-разработки (SP).

Сигналы: Claude Sonnet 5, AG-UI protocol, Managed entitlements

AutoGen: В сообществе AutoGen появились open-source библиотеки для security testing (175 тестов для агентов) и identity management (proveyouragent с Ed25519 токенами) (DP). Это усиливает критерии Secrets/tenant и Prod-ops, так как решает проблемы аутентификации и аудита в multi-agent системах (CP). Для SL2 AI Platform это давление на GAP G3 (нет inter-agent distributed tracing) и G9 (молодой проект без community), так как AutoGen становится более зрелым в части безопасности и observability (SP).

Сигналы: Security testing, Identity management

AgentForce: Salesforce представил continuous planning systems и детерминированные conversational flows с Agent Script (DP). Это усиливает критерий Code+No-code, так как Agent Script позволяет бизнес-пользователям создавать предсказуемые workflows без кода (CP). Для SL2 AI Platform это давление на GAP G1 (нет visual builder), так как AgentForce предлагает более низкий порог входа для не-разработчиков (SP).

Сигналы: Continuous planning systems, Agent Script

n8n: Вышел релиз с поддержкой self-improving agent skills через cognee и AI maturity framework (DP). Это усиливает критерий Code+No-code, так как позволяет автоматически улучшать навыки агентов без ручного вмешательства (CP). Для SL2 AI Platform это давление на GAP G7 (молодой проект без marketplace), так как n8n развивает экосистему готовых интеграций (SP).

Сигналы: Self-improving agent skills, AI maturity framework
