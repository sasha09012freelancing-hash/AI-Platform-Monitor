3. Детали по платформам

Vertex AI: Google активно развивает экосистему, интегрируя LLM непосредственно в SQL-запросы баз данных и улучшая инструменты для медицинских приложений. Платформа укрепляет позиции как Enterprise-решение с упором на аналитику данных.

Сигналы: Запуск управляемого сервиса Airflow, признание лидером Gartner. Ссылка

AWS Bedrock: AWS сосредоточилась на безопасности AI-агентов, введя гранулярный контроль доступа к данным на уровне документов. Значительное внимание уделено real-time voice-агентам и интеграции с корпоративными браузерами для контроля контура.

Сигналы: Контроль доступа в Quick, интеграция с Cisco AI Defense. Ссылка

Azure Foundry: Происходит стремительное обновление стека: релиз Foundry Local 1.1 и поддержка широкого спектра новых моделей, включая GPT-5.5. Упор сделан на локальное развертывание и инструменты для VS Code.

Сигналы: Релиз Foundry Local 1.1, интеграция новых моделей GPT и Claude. Ссылка

AutoGen: Сообщество активно разрабатывает методы защиты памяти агентов и системы оценки доверия для предотвращения отравления данных. Появились интересные архитектурные концепты по замене чат-историй на протоколы долговечных рабочих пространств.

Сигналы: Инструмент OWASP для защиты памяти, концепт AI Workroot. Ссылка

AgentForce: Salesforce трансформирует разработку агентов, предлагая человекочитаемые скрипты и продвинутые инструменты для отладки визуальных графов выполнения. Акцент на "агент-первом" подходе с использованием MCP для вызова Apex-логики.

Сигналы: Новый жизненный цикл метаданных, инструмент отладки AgentLens. Ссылка

LangGraph: Произошел переход от альфа-версий к официальным релизам стабильных веток, что фиксирует API и готовит базу для продакшн-масштабирования. Улучшена поддержка стриминга событий для агентских систем.

Сигналы: Выход LangGraph 1.2.0 и LangChain 1.3.0. Ссылка

n8n: Получение инвестиций от SAP и встраивание в Joule Studio превращает n8n в стандарт визуальной автоматизации для enterprise-сегмента. Это прямое конкурентное давление на no-code составляющую других платформ.

Сигналы: Инвестиции SAP в $5.2 млрд, интеграция в SAP Joule Studio. Ссылка

5. Выводы для команды

  1. Конкуренты смещают фокус на визуализацию отладки (AgentLens, n8n) — нам необходимо усилить Dashboard мониторинга выполнения агентов в SL2 AI Platform.
  2. Безопасность и "гигиена" памяти агентов (OWASP Agent Memory Guard) становятся критическим требованием для enterprise — следует рассмотреть внедрение встроенных механизмов проверки входящих данных в SL2 AI Platform.
  3. Интеграция n8n в SAP делает их очень опасным конкурентом в Enterprise No-code. Необходимо подчеркнуть наше превосходство в Self-hosted архитектуре и контроле данных, где n8n проигрывает из-за зависимости от экосистемы SAP.
